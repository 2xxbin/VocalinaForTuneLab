# VocalinaForTuneLab

티젠스의 음성합성엔진 `보카리나`를 오픈소스 에디터 [`TuneLab`](https://github.com/LiuYunPlayer/TuneLab)으로 이식한 프로그램입니다.

> 본 프로그램은 티젠스의 명시적 허락 하에 개발되었으며, 소스코드 비공개 조건으로 배포 협의가 완료되었습니다. <br>
> 모든 저작권은 티젠스에게 있으며, 본 프로그램을 허용된 사용 범위를 벗어난 역설계, 소스 분석, 데이터 추출 등의 행위에 사용할 수 없습니다. <br>
> 해당 프로그램을 원작자의 허락 없이 무단 복제, 배포, 수정할 수 없습니다. <br>
> 해당 프로그램은 언제든 사전 고지 없이 내려갈 수 있습니다.

해당 프로그램은 64 비트 윈도우 전용이며, 32비트 혹은 기타 OS에서의 동작은 보장하지 않습니다.
